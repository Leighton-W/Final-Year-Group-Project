# Final-Year-Group-Project

A 3D open-world game developed in Unity as part of my final-year Games Design &amp; Production degree project.
The project was developed by a team of five using Agile methodologies including sprint planning, scrum meetings and iterative development.

## My Responsibilities
- Primary gameplay programmer
- Implemented player movement and combat systems
- Developed interaction systems
- Tested and debugged gameplay features
- Collaborated with designers and developers throughout development

## Technologies Used
- Unity
- C#
- Version Control

## What I Learned

This project gave me experience working within a multidisciplinary development team while contributing to a larger codebase. I developed my technical skills in C#, gameplay programming, debugging and iterative development while learning how to communicate effectively within an Agile workflow.
